## 🎬 3 manières d'utiliser cette vidéo

### Option 1 — Diffusion directe sur le site

La vidéo est déjà sur votre serveur après déploiement.
**URL publique** : `https://veritas-school.com/VERITAS_Promo_Video.html`

Vous pouvez :
- **Partager le lien sur WhatsApp** → le contact ouvre la vidéo dans son navigateur
- **L'intégrer dans la page d'accueil** : déjà fait via la vidéo héro
- **L'envoyer par email** comme présentation

### Option 2 — Convertir en MP4 (recommandé pour Facebook/Instagram)

#### 🔧 Méthode A — OBS Studio (PC, gratuit, qualité pro)

1. **Installer OBS Studio** : https://obsproject.com/ (100% gratuit)
2. **Ouvrir la vidéo HTML** dans Chrome ou Edge en plein écran (F11)
3. **Lancer OBS** :
   - Ajouter une "Source" → **Capture d'écran**
   - Sélectionner le moniteur où joue la vidéo
4. **Régler la qualité** :
   - Paramètres → Sortie → Résolution : **1920×1080**
   - Bitrate : **6000 Kbps** (HD propre)
   - Encodeur : **NVENC** (si carte NVIDIA) ou **x264**
   - Format de sortie : **MP4**
5. **Cliquer "Démarrer l'enregistrement"** dans OBS
6. **Cliquer "▶ Lancer la présentation"** dans Chrome
7. **Attendre 3 min 06s** — laissez la vidéo défiler
8. **Cliquer "Arrêter l'enregistrement"** quand "🎬 Fin de la vidéo" apparaît
9. ✅ Votre `VERITAS_Promo.mp4` est dans `Vidéos/`

⏱️ **Temps total** : 15 minutes (installation + 1 enregistrement)

#### 🔧 Méthode B — Loom (Web, gratuit, le + simple)

1. Aller sur https://www.loom.com/ → créer un compte gratuit
2. Installer l'extension Chrome **Loom**
3. Ouvrir `VERITAS_Promo_Video.html` en plein écran (F11)
4. Cliquer sur l'icône Loom dans Chrome → **« Record screen »**
5. Sélectionner « Onglet actuel uniquement » → **Démarrer**
6. Cliquer « ▶ Lancer la présentation » dans la vidéo
7. Attendre la fin → Arrêter
8. Loom génère un **lien partageable instantané** + un **MP4 téléchargeable**

⏱️ **Temps total** : 5 minutes

#### 🔧 Méthode C — Capture mobile (Android/iPhone)

1. Ouvrir `https://veritas-school.com/VERITAS_Promo_Video.html` sur votre téléphone
2. Activer l'**enregistrement d'écran** (Centre de contrôle iOS / Quick Tile Android)
3. Mettre en plein écran (rotation paysage)
4. Cliquer "▶ Lancer la présentation"
5. Attendre 3 minutes → arrêter
6. La vidéo MP4 est dans votre galerie

⏱️ **Temps total** : 5 minutes (qualité 1080p garantie sur smartphones récents)

## 💡 Astuces qualité

- ✅ **Plein écran obligatoire** avant d'enregistrer (F11) — sinon les barres de Chrome apparaissent
- ✅ **Désactiver les notifications Windows** pendant l'enregistrement (Action Center → Concentration)
- ✅ **Désactiver le pointeur de souris** dans OBS : Sources → Capture d'écran → Propriétés → décocher "Capturer le curseur"
- ✅ **Fermer les autres onglets/applis** pour éviter les ralentissements
- ✅ **Brancher l'ordinateur sur secteur** pour éviter l'écomode CPU
- ✅ **1080p suffit largement** — 4K c'est inutile pour Facebook/WhatsApp

---

## 🆘 Problèmes fréquents

| Problème | Solution |
|---|---|
| Vidéo saccadée pendant l'enregistrement | Fermer onglets/applis ; baisser bitrate OBS à 4000 Kbps |
| Texte flou dans le MP4 | Vérifier que la résolution OBS = 1920×1080 ; pas de scaling |
| Animations qui sautent une frame | Augmenter FPS OBS à 60 ; mettre Chrome en mode "Haute performance" |
| Pas de son dans le MP4 | OBS → Mixeur audio : activer "Audio Bureau" |
| Fichier trop gros (> 100 Mo) | Compresser avec HandBrake (gratuit) → Preset "Web Optimized" |
